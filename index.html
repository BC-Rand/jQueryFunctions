.click, .hide, and .show

When you click the button, this text will become hidden.


.toggle

When you click the button, this text will toggle from hidden to visible

.slideDown and .slideUp


This text will slide up to a hidden position when the .slideUp button is pressed and move down from the hidden position when .slideDown is pressed.

.slideToggle

This text will appear aor disappear with a sliding animation when the button is pressed.

Fade in and Fade out

When you click the button, this text will fade out or in.


.addClass

This text will gain the 'red' class when the button is pressed, applying the .red CSS styling

.before

Text will be inserted before this paragraph when the button is pressed.

.append

When the button is pressed, additional text will be added to the end of this p tag.

.html

When the button is pressed this text will be replaced in the HTML code with new content.

.attr

When the button is pressed it will add text of the classes of this p tag

.val

The lower text will display the value of the selector under the button


.data

When .data Retrieve is pressed, the "Data" value from the #datatextdiv will be displayed. .data Set Yes and .data Set Maybe will set the "Data" value in #datatextdiv to Yes and Maybe respectively. .data Clear will clear the "Data" value.
